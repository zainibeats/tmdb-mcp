# TMDB Embed URL Generator

A web interface for generating streaming embed URLs using TMDB IDs with support for movies and TV shows.

## ⚠️ Educational Purpose Only

**This tool is strictly for educational purposes only.** It demonstrates web development concepts and API integration techniques. Users are responsible for complying with all applicable laws and terms of service when using generated URLs.

## Quick Start

1. **Launch with Docker**:
   ```bash
   docker run -d --name tmdb-embed-ui -p 8689:8689 skimming124/tmdb-embed-ui:latest
   ```

2. **Access the interface** at http://localhost:8689

Clickable URLs generated by the MCP resolver also work directly:

```text
http://localhost:8689/?mediaType=movie&tmdbId=550
http://localhost:8689/?mediaType=tv&tmdbId=1396&season=1&episode=1
```

These links pre-fill the form and automatically generate provider URLs.

## Usage

1. Select media type (Movie/TV Show)
2. Enter TMDB ID (e.g., 550 for Fight Club)
3. For TV shows: add season and episode numbers
4. Choose streaming providers
5. Click "Generate Embed URLs" and copy results

## Supported Providers

See providers [here](src\providers.json)

## Configuration

Customize providers in `providers.json`:

```json
{
  "providers": [
    {
      "name": "ProviderName",
      "movie": "https://example.com/movie/{id}",
      "tv": "https://example.com/tv/{id}/{season}/{episode}"
    }
  ]
}
```

**Variables**: `{id}` (TMDB ID), `{season}`, `{episode}` (TV only)
